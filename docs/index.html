DeepAR.ai - face mask AR library for Android - v1.4.5

DeepAR.ai is a multiplatform (Android/iOS) augmented reality library that provides applications with 3D face masks and other Snapchat-like effects like filters, lenses, animations etc.

The trial version can only be run for limited time

Key features

DeepAR.ai Android standard SDK provides following features:

  • Applying 3D models to the face (rigid or deformable)
  • Filters
  • Visual effects (i.e. breathing fire)
  • Face morphing
  • Screenshot (front and back camera)
  • Video capture (front and back camera)
  • Adding sound effects for recorded videos (currently not supported)
